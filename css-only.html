CSS-only examples

The following examples are constructed using only markup and the kumquat.css file.

Minimal example

jade / pug template:

.kumquat-terminal
    header
      .btn.green
      .btn.yellow
      .btn.red
      .title
    section.terminal
      .cli
        .line
          .cursor

emmet expression:

.kumquat-terminal>(header>.btn.green+.btn.yellow+.btn.red+.title{kumquat example})+(section.terminal>.cli>.line{type}>span.cursor))
kumquat example
