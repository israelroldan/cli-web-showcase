CSS-only examples

The following examples are constructed using only markup and the kumquat.css file.

Minimal example

jade / pug template:

.kumquat-terminal
    .header
      .btn.green
      .btn.yellow
      .btn.red
      .title
    .terminal
      .cli
        .line.with-prompt
          .cursor

emmet expression:

.kumquat-terminal>(.header>.btn.green+.btn.yellow+.btn.red+.title{kumquat example})+(.terminal>.cli>.line.with-prompt{type}>span.cursor)
kumquat example
